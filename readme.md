## BING 壁纸 API 用法  
 - 1. 访问 http://api.ioliu.cn/bing , 返回bing每日最新背景图.  
 - 2. 访问 http://api.ioliu.cn/bing?d=n (n>=0), 返回以当日为起点第n天前的壁纸.  
 - 3. 访问 http://api.ioliu.cn/bing/json , 返回bing每日最新壁纸的相关(介绍、图片地址等)信息(json格式)..
